### 项目介绍
本系统旨在构建一个智能农业平台，通过整合传感器、执行器和智能算法，实现对农业生产过程的监控、管理和优化。

### 项目背景
传统的农业生产方式依赖人工操作，效率低、成本高。而智能农业平台通过引入传感器、执行器和智能算法，实现对农业生产过程的实时监控和管理，提高生产效率和资源利用率。

### 项目目标
- 实现对农业生产过程的实时监控，包括土壤湿度、温度、光照等环境参数。
- 提供农业生产管理功能，如自动灌溉、施肥、监控作物生长状态等。
- 利用智能算法优化农业生产过程，提高生产效率和资源利用率。

### 项目架构
采用pnpm + monorepo 架构，将项目拆分成多个子模块，每个子模块负责不同的功能。
- 前端：Vue.js + PrimeVue
- 后端：Node.js + Nest.js
- 数据库：PostgreSQL + postgis
